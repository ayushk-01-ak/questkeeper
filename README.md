# QuestKeeper 🎲

A local AI-powered Dungeon Master and NPC companion.

Built with Python, Ollama, FastAPI, Streamlit, and ChromaDB.
Runs completely offline using a local LLM.

## Project Status
🚧 Phase 0 — Environment Setup (In Progress)

## Tech Stack
- Python 3.11
- Ollama (local LLM)
- FastAPI (backend)
- Streamlit (frontend)
- ChromaDB (vector database)
- SQLite (character/session storage)

## Setup Instructions
1. Clone this repository
2. Create virtual environment: `py -3.11 -m venv venv`
3. Activate it: `venv\Scripts\activate`
4. Install dependencies: `pip install -r requirements.txt`
